##### 查阅命令帮助信息

+ --help

  ```
  command --help
  ```

+ man

  ```
  man command
  ```

+ 使用man查看帮助信息时的一些操作键

  | 操作键  |         功能         |
  | :-----: | :------------------: |
  | 空格键  |  显示手册页的下一屏  |
  | Enter键 | 一次滚动手册页的一行 |
  |    b    |       回滚一屏       |
  |    f    |       前滚一屏       |
  |    q    |         退出         |
  |  /word  |    搜索word字符串    |
