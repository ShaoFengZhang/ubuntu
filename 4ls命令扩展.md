



##### ls命令扩展

+ ls命令的扩展
  + ls类似于dos命令下的dir
  + ls -a :可以查看隐藏目录
  + ls -l:显示文件和文件夹的详细信息
  + ls -l -h:配合 -l命令使用
  + ls -lha

+ ls通配符的使用

  | 通配符 |              含义              |
  | :----: | :----------------------------: |
  |   *    |         代表任意个字符         |
  |   ？   |   代表任意一个字符，至少一个   |
  |   []   | 表示可以匹配字符组中的任意一个 |
  | [abc]  |      匹配abc中的任意一个       |
  | [a-f]  |   匹配从a-f范围内的任意一个    |
